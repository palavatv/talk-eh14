Signaling Protocol

A few defined JSON messages

Client → Server


join_room(room_id, status)

send_to_peer(peer_id, data)

update_status(status)

Server → Client


joined_room(me, peers)

peer_updated_status(peer_id, peer_status)

error(id, message)

shutdown(seconds)

Client → Client

via send_to_peer
